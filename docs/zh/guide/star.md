## 为什么要 Star？

- **让更多人看到** — Star 越多，越容易被其他开发者和团队发现
- **激励团队** — 每一颗 Star 都是我们持续迭代的动力
- **方便找回** — Star 过的项目会出现在你的 GitHub Stars 列表中

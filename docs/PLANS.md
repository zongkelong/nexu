# Plans

Execution planning is tracked in `exec-plans/`.

| Location | Purpose |
|----------|---------|
| `exec-plans/active/` | Plans currently being executed |
| `exec-plans/completed/` | Finished plans (kept for reference) |
| `exec-plans/tech-debt-tracker.md` | Ongoing technical debt log |

## How to use

1. Write a plan as a markdown file with clear steps, acceptance criteria, and dependencies.
2. Place it in `exec-plans/active/`.
3. Move to `exec-plans/completed/` when done.
4. Log any debt discovered during execution in `tech-debt-tracker.md`.
